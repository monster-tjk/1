Тарҳи Вебсайти Cootels (Лэндинг)
Ин лоиҳа тарҳи саҳифаи асосии (Landing Page) вебсайти Cootels мебошад, ки барои брон кардани хонаҳо ва истироҳатгоҳҳо дар табиат пешбинӣ шудааст. Тарҳ бо услуби муосир ва тоза сохта шудааст.

Хусусиятҳои тарҳ
Қисми болоӣ (Header): Сарлавҳа бо логотипи лоиҳа ("Cootels"), менюи пайвандҳо (Rooms, Reservation, Contacts) ва тугмаи "Get Started".

Қисми асосӣ (Hero Section): * Шиори асосӣ: "Nature, Warmth, and Beauty in One Space" (Табиат, гармӣ ва зебоӣ дар як ҷой).

Матнии кӯтоҳ дар бораи рафъи стресс ва истироҳат дар табиат.

Тугмаи фаъоли "Reservation" (Бронкунӣ).

Қисми визуалӣ: Се акси амудии ба ҳам пайваст, ки намуди хонаи чӯбини зебоеро дар байни ҷангал ва барф нишон медиҳанд.

Технологияҳои тавсияшаванда барои сохтан
Барои коди ин тарҳ, шумо метавонед технологияҳои зеринро истифода баред:

HTML5 — барои сохтори саҳифа.

CSS3 (ё Tailwind CSS / SASS) — барои ороиш ва ҷойгиркунии унсурҳо (Flexbox ё CSS Grid барои қисми аксҳо).

JavaScript — барои кор кардани тугмаҳо ва меню дар телефонҳо.

Чӣ тавр истифода бурдан
Аксҳоро аз тарҳ бурида, дар ҷузвдони (папкаи) images/ нигоҳ доред.

Файли index.html созед ва сохтори саҳифаро нависед.

Стилҳоро дар файли style.css илова кунед, то тарҳ ба акс монанд шавад.
